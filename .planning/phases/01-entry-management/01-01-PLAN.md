<task type="auto">
  <name>Task 2: Add update, soft delete, restore, and cleanup methods to StorageService and ReflectionService</name>
  <files>lib/services/storage_service.dart, lib/services/reflection_service.dart</files>
  <action>
**StorageService changes:**

1. Add `updateEntry(Entry entry)` method:
   - Creates a new Entry with all fields from the passed entry but `updatedAt: DateTime.now()`
   - Calls `_box.put(entry.id, updated)` to overwrite in Hive
   - Returns void

2. Add `softDeleteEntry(String entryId)` method:
   - Gets entry from `_box.get(entryId)`, returns if null
   - Creates new Entry with `isDeleted: true`, `deletedAt: DateTime.now()`, `updatedAt: DateTime.now()`
   - Calls `_box.put(entryId, deleted)`

3. Add `restoreEntry(String entryId)` method:
   - Gets entry from `_box.get(entryId)`, returns if null or not deleted
   - Creates new Entry with `isDeleted: false`, `deletedAt: null`, `updatedAt: DateTime.now()`
   - Calls `_box.put(entryId, restored)`

4. Add `_permanentDeleteOldEntries()` private method:
   - Finds all entries where `isDeleted == true` AND `deletedAt` is before `DateTime.now().subtract(Duration(days: 30))`
   - For each: delete associated ReflectionAnswers via `ReflectionService.instance.deleteAnswersForEntry(entry.id)`
   - Then `_box.delete(entry.id)`
   - Called during `init()` after box opening (add call after `_backfillLongestStreak()`)

5. Update `getAllEntries()`:
   - Add `.where((entry) => !entry.isDeleted)` before `.toList()`

6. Update `getEntriesForDate(DateTime date)`:
   - Add `&& !entry.isDeleted` to the where clause

7. Update `getEntriesForPeriod(DateTime start, DateTime end)`:
   - Add `&& !entry.isDeleted` to the where clause

8. Add `getEntry(String id)` method:
   - Returns `_box.get(id)` (nullable Entry?, does NOT filter by isDeleted — needed for restore/undo flow)

9. Add import for ReflectionService at top of file:
   `import 'reflection_service.dart';`

**ReflectionService changes:**

1. Add `updateAnswer({required String answerId, required String newAnswerText})` method:
   - Gets answer from `_answers.get(answerId)`, returns if null
   - Creates new ReflectionAnswer with same id, entryId, questionId, questionText, createdAt but `answer: newAnswerText`
   - Calls `_answers.put(answerId, updated)`

2. Add `deleteAnswer(String answerId)` method:
   - Calls `_answers.delete(answerId)`

3. Add `deleteAnswersForEntry(String entryId)` method:
   - Finds all answers where `answer.entryId == entryId`
   - Deletes each from `_answers`
   - Used by StorageService during permanent delete cleanup
