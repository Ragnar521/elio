**Widget props:**
```dart
class CalendarHeatmap extends StatelessWidget {
  const CalendarHeatmap({
    super.key,
    required this.month,              // DateTime representing displayed month (year + month only)
    required this.entriesByDate,      // Map<DateTime, List<Entry>> grouped by date-only keys
    required this.onDayTap,           // Function(DateTime date, List<Entry> entries) for day tap
    required this.onMonthChanged,     // Function(int offset) — +1 for next, -1 for prev
    this.selectedDate,                // DateTime? — currently highlighted day (accent ring)
    this.canNavigateBack = true,      // Whether back arrow is enabled
    this.canNavigateForward = true,   // Whether forward arrow is enabled
  });
}
```

**Build method structure — return a Column with these children in order:**

1. **Month navigation header:**
   - A `Row` with: left `IconButton` (chevron_left), center `Expanded` `Text` (month + year label, e.g. "February 2026"), right `IconButton` (chevron_right)
   - Left arrow: `onPressed` is `canNavigateBack ? () => onMonthChanged(-1) : null`
   - Right arrow: `onPressed` is `canNavigateForward ? () => onMonthChanged(1) : null`
   - Disabled arrows use `ElioColors.darkPrimaryText.withOpacity(0.3)`, enabled use full `ElioColors.darkPrimaryText`
   - Month label style: `Theme.of(context).textTheme.titleMedium` with `fontWeight: FontWeight.w600`
   - `SizedBox(height: 12)` after header

2. **Weekday labels row:**
   - A `Row` with 7 equally spaced `Text` widgets: 'M', 'T', 'W', 'T', 'F', 'S', 'S'
   - Each wrapped in `Expanded` with `Center` alignment
   - Style: `TextStyle(fontSize: 12, fontWeight: FontWeight.w600, color: ElioColors.darkPrimaryText.withOpacity(0.5))`
   - `SizedBox(height: 8)` after labels

3. **Calendar grid:**
   - Wrap the grid in a `GestureDetector` for swipe handling:
     ```dart
     onHorizontalDragEnd: (details) {
       final velocity = details.primaryVelocity ?? 0;
       if (velocity.abs() < 200) return;
       if (velocity < 0 && canNavigateForward) onMonthChanged(1);
       if (velocity > 0 && canNavigateBack) onMonthChanged(-1);
     }
     ```
   - Use a `Column` of `Row` widgets (NOT GridView — only 35-42 cells, no scrolling needed)
   - Build the list of days using `_buildCalendarDays(month)` (see below)
   - Chunk the days list into rows of 7
   - Each row is a `Row` with 7 `Expanded` children wrapping day cells
   - Add `SizedBox(height: 4)` between rows (the 4px gap)

4. **Color legend:**
   - `SizedBox(height: 12)` before legend
   - A centered `Row` with: "Low" text, 8px gap, gradient `Container` (width: 120, height: 8, gradient from #4B5A68 to #FF6436, borderRadius: 4), 8px gap, "High" text
   - Text style: `TextStyle(fontSize: 11, color: ElioColors.darkPrimaryText.withOpacity(0.5))`

**Private helper methods:**

`List<DateTime?> _buildCalendarDays(DateTime month)`:
- Calculate first day: `DateTime(month.year, month.month, 1)`
- Calculate last day: `DateTime(month.year, month.month + 1, 0)`
- Leading empty cells: `firstDay.weekday - 1` (Monday = 1)
- Add `null` for each leading empty cell
- Add `DateTime(month.year, month.month, day)` for each day 1..lastDay.day
- Add trailing `null` entries until `days.length % 7 == 0`
- Return the list

`Widget _buildDayCell(BuildContext context, DateTime? date)`:
- If `date == null`: return `const SizedBox.shrink()` (wrapped in Expanded by parent)
- Look up entries: `final entries = entriesByDate[DateTime(date.year, date.month, date.day)] ?? []`
- `final hasEntries = entries.isNotEmpty`
- Calculate average mood: `final avgMood = hasEntries ? entries.fold(0.0, (sum, e) => sum + e.moodValue) / entries.length : 0.0`
- Determine if today: compare date with `DateTime.now()` (year, month, day)
- Determine if selected: compare with `selectedDate` (year, month, day)
- Background color: `hasEntries ? _moodColor(avgMood) : ElioColors.darkSurface.withOpacity(0.3)`
- Border logic:
  - If today AND selected: `Border.all(color: ElioColors.darkAccent, width: 2.5)`
  - If today: `Border.all(color: ElioColors.darkAccent, width: 2)`
  - If selected: `Border.all(color: ElioColors.darkAccent, width: 1.5)`
  - Otherwise: `null`
- Text color: `hasEntries ? Colors.white : ElioColors.darkPrimaryText.withOpacity(0.35)`
- Text weight: today ? `FontWeight.w700` : `FontWeight.w400`
- Return `GestureDetector` wrapping a `Container`:
  - `GestureDetector.onTap`: `hasEntries ? () => onDayTap(date, entries) : null`
  - Container: `height: 40`, no fixed width (uses Expanded from parent), `decoration` with background color, border, `BorderRadius.circular(8)`
  - Child: `Center(child: Text('${date.day}', style: ...))`

`Color _moodColor(double value)`:
- `const low = Color(0xFF4B5A68);`
- `const high = Color(0xFFFF6436);` (ElioColors.darkAccent value)
- `return Color.lerp(low, high, value) ?? high;`

`String _monthLabel(DateTime month)`:
- Use a list of full month names: January, February, etc.
- Return `'${names[month.month - 1]} ${month.year}'`

`bool _isSameDay(DateTime a, DateTime b)`:
- `return a.year == b.year && a.month == b.month && a.day == b.day;`
