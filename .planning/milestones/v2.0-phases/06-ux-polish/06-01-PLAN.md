**2. Create `lib/widgets/animated_tap.dart`:**

A reusable wrapper that adds subtle scale animation + optional haptic feedback to any widget.

```dart
import 'package:flutter/material.dart';
import 'package:flutter/services.dart';

class AnimatedTap extends StatefulWidget {
  const AnimatedTap({
    super.key,
    required this.child,
    this.onTap,
    this.pressScale = 0.97,
    this.enableHaptic = false,
  });

  final Widget child;
  final VoidCallback? onTap;
  final double pressScale; // 0.97 for buttons, 0.98 for cards
  final bool enableHaptic;

  @override
  State<AnimatedTap> createState() => _AnimatedTapState();
}
```

Implementation details:
- Use GestureDetector with onTapDown/onTapUp/onTapCancel for press state tracking
- Use AnimatedScale with 150ms duration and Curves.easeInOut
- On tap: if enableHaptic is true, call HapticFeedback.lightImpact()
- Boolean `_isPressed` state controls scale: `_isPressed ? widget.pressScale : 1.0`
- Wrap haptic call in try-catch for platform safety (per research pitfall 3)
