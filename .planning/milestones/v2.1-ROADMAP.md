# Milestone v2.1: Demo Mode

**Status:** SHIPPED 2026-02-28
**Phases:** 7-9
**Total Plans:** 4

## Overview

Add demo/showcase mode with realistic sample data so the app can be demonstrated with a full, lived-in feel. Includes a launcher screen for choosing between demo and fresh start, and a triple-tap reset mechanism.

## Phases

### Phase 7: Sample Data Engine

**Goal**: App has realistic sample data that showcases all features
**Depends on**: Phase 6 (completed)
**Requirements:** DATA-01, DATA-02, DATA-03, DATA-04, DATA-05
**Plans**: 2 plans

Plans:
- [x] 07-01: Core sample data service (entries, reflections, directions, connections, streaks)
- [x] 07-02: Weekly summaries + visual verification

**Details:**
- Created SampleDataService with deterministic seeding (seed=42,43,44)
- ~90 days of entries with realistic mood patterns (lower Mondays, higher weekends, gaps)
- 3 directions (Career, Health, Relationships) + Peace with uneven distribution
- Direct Hive box writes for backdated timestamps
- ~12 weeks of weekly summaries with trends and takeaways

### Phase 8: Launcher Screen

**Goal**: Users can choose between sample data demo and fresh start before onboarding
**Depends on**: Phase 7
**Requirements:** LAUNCH-01, LAUNCH-02, LAUNCH-03
**Plans**: 1 plan

Plans:
- [x] 08-01: Launcher screen UI + app gate wiring

**Details:**
- LauncherScreen with two clear options (demo data / fresh start)
- Gate logic in OnboardingGate via launcher_completed setting
- Demo mode sets both launcher_completed and onboarding_completed
- Loading overlay during demo data generation

### Phase 9: Reset Mechanism

**Goal**: Users can wipe all data and return to launcher for re-demonstration
**Depends on**: Phase 8
**Requirements:** RESET-01
**Plans**: 1 plan

Plans:
- [x] 09-01: Triple-tap reset to wipe all data and return to launcher

**Details:**
- Triple-tap Home icon triggers full data wipe
- wipeAllData() clears all 7 Hive boxes (settings last to prevent partial state)
- Navigation resets to OnboardingGate to leverage existing gate logic
